## Proposed Changes

We will create a new directory named `portfolio-website` on your Desktop. Inside, we will organize our code cleanly into three main files.

### HTML Structure
We will create `index.html`. This will be the skeleton of our website, containing semantic HTML5 tags for SEO-friendly structure.
#### [NEW] `index.html`
- Contains sections: `<header>`, `<section id="home">`, `<section id="about">`, etc.
- SEO meta tags included.

### Styling (CSS)
We will create `style.css`. This file will handle the visual presentation.
#### [NEW] `style.css`
- Modern aesthetics using CSS variables for easy Dark/Light mode switching.
- Flexbox and CSS Grid for responsive layout (mobile-friendly).
- Keyframe animations for smooth transitions and hover effects.

### Interactivity (JavaScript)
We will create `script.js`. This will add dynamic behavior.
#### [NEW] `script.js`
- Logic for toggling Dark/Light mode.
- Smooth scrolling for navigation links.
- Animating skill progress bars when they scroll into view.
